# Education App styling Practice

## 소개

SCSS문법과 BEM 네이밍으로 flexbox & Grid 프로퍼티를 적절히 활용해 layout 연습하기 위한 클론 작업물.

## 주요 기능

[vite] [SASS]

```js
// Vite install
npm create vite@latest
//  Sass install
npm add -D sass
```
